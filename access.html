Our intention when making our website, was to make the site easily accessible and user friendly for everyone. We have made legible subpages that everyone can use and a straight-forward booking form. When creating the website, we wanted it to look simple and legible. We checked the usability of the code on the website: Validator. The automatic test helped us by pointing out some minor errors and some potential improvements in the code to make it even more readable. An example of this is that our backgrounds and text did not have enough contrast between them, and as a result we made our backgrounds darker. We wanted to use as little code as possible so as not to lag the page.
